# samples/ — レシート画像配置場所

このディレクトリには、講座ハンズオン用のレシート画像を配置します。

## ファイル配置

- 拡張子：`.jpg` / `.jpeg` / `.png`
- 命名：`receipt_NNN.jpg`（連番推奨。例: `receipt_001.jpg`）
- 配置数：講座中は3〜5枚あれば十分

## ⚠ 個人情報の取り扱い（重要）

**このディレクトリ配下の画像ファイルは `.gitignore` で commit 対象外になっています。**

- 自分のレシートを使う場合：個人情報（氏名・住所・電話番号・カード番号下4桁等）が写っていないか必ず確認
- 公開リポジトリには絶対に画像を含めない
- **万が一 commit してしまった場合：** `git rm --cached samples/*.jpg && git commit && git push --force` で履歴から完全削除

## 講師による配布画像

講座当日は、講師から共有された**架空のサンプルレシート画像**を使うことも可能です。
配布された画像は本人の個人情報を含まないため、安心してハンズオンに使えます。

## このディレクトリ配下で唯一 commit される唯一のファイル

- `samples/README.md`（このファイル）

それ以外の画像ファイルは `.gitignore` の `samples/*` ＋ `!samples/README.md` 例外設定により Git 管理対象外です。
