# 経費科目判定スキル（5分類）

このスキルは、レシートから読み取った情報を **会議費／消耗品費／旅費交通費／新聞図書費／雑費** のいずれかに分類する。**この5つ以外を勝手に追加してはならない。**

## 判定の優先順位（重要）

複数科目に該当しうる場合、以下の順で判定する：

1. **店名・業種から明確に推定できる科目**（例：書店 → 新聞図書費）
2. **品目の主たる用途**（例：100円ショップで茶菓子のみ → 会議費）
3. **判別不能なら雑費**（無理に他科目に寄せない）

迷った時の最終ルール：**「迷ったら雑費」ではなく、まず4分類のどれかに該当しないか再検討してから雑費に落とす**。

---

## 1. 会議費

社内・社外の打合せに関連する飲食代・茶菓子代。**自分1人の食事は対象外**。

該当例：
- カフェ・喫茶店・ファストフード（スターバックス／ドトール／マクドナルド等）で複数名利用が示唆されるもの
- 取引先・社内メンバーとの飲食店利用（居酒屋・レストラン）
- 打合せ用の手土産・差し入れ・会議用茶菓子（和菓子店・洋菓子店）
- 会議室レンタル・コワーキングスペースのドロップイン利用料
- 出前・デリバリー（複数名分の量・複数の弁当が含まれる場合）

判定ヒント：
- スターバックス1杯のみ → 会議費（打合せの可能性が高い前提で判定）
- スーパーで茶菓子（個包装の菓子・お茶）大量購入 → 会議費

## 2. 消耗品費

業務で使う文房具・小物備品・PC周辺の消耗品。**食料品は含めない**。

該当例：
- 文具（ノート・ボールペン・付箋・ファイル・テープ類）
- PC・オフィス小物（USBメモリ・マウス・ケーブル・電池・プリンタ用紙）
- 100円ショップでの雑貨購入（キャンドゥ・ダイソー・セリア）
- 清掃用品・トイレットペーパー等のオフィス備品
- 印刷費（コンビニコピー・キンコーズ）

判定ヒント：
- スーパーでの食料品・日用品 → **雑費**（消耗品費にはしない）
- 100円ショップで食品（飲料・お菓子）のみ → **会議費**（茶菓子扱い）

## 3. 旅費交通費

業務上の移動・出張に関わる交通費・宿泊費。

該当例：
- 鉄道（JR・私鉄・地下鉄）の切符・IC チャージ・回数券
- タクシー・ハイヤー・配車サービス（GO・Uber 等）
- バス・高速バス・空港リムジン
- ETC・ガソリン代・駐車場代・コインパーキング
- 出張時の宿泊費（ホテル・ビジネスホテル）

判定ヒント：
- レンタカー → 旅費交通費
- 高速道路 SA/PA での飲食 → **会議費**（複数名）または **雑費**（自分の食事）

## 4. 新聞図書費

業務に関連する書籍・新聞・雑誌・電子書籍購読料。

該当例：
- 書店（紀伊國屋・丸善・ジュンク堂・蔦屋）での書籍購入
- Amazon・楽天ブックスでの書籍・電子書籍
- 新聞購読料（日経・朝日・読売）・業界紙
- 雑誌・専門誌・有料コラム（note の有料記事を含む）
- オンライン辞書・データベース利用料（業務関連のもの）

判定ヒント：
- Kindle Unlimited 等の月額サブスク → 新聞図書費
- 漫画・小説（業務関連性が薄い） → **雑費**

## 5. 雑費

上記4分類のいずれにも明確に当てはまらないもの。

該当例：
- 自分1人の食事（弁当・コンビニ食・スーパーの惣菜）
- スーパー・ドラッグストアでの食料品・日用品購入
- 銀行手数料・振込手数料
- クリーニング代・名刺発注（業務用）
- 慶弔費・少額の交際費（会議費に分類できない場合）

## 判定の難しい3例の解説

実運用で AI が迷いがちな代表例。**この3つの判断基準を覚えておくと精度が上がる**。

### ケース1：マルエツ（スーパー）で 700 円

**誤判定しがちな分類：消耗品費**

スーパーは食料品・日用品・雑貨が混在するため、品目が見えないレシートでは「日用品 → 消耗品費」と推論してしまう。しかし本ルールでは：

- スーパー＝食料品中心 → **雑費**
- 食料品・日用品の購入は消耗品費に含めない（消耗品費は文房具・PC小物・オフィス備品に限定）

**判定：雑費**

例外：明らかに「会議用の茶菓子・大量の個包装菓子・ペットボトル飲料の箱買い」等が読み取れる場合は **会議費**。

### ケース2：キャンドゥ（100円ショップ）で 665 円

**誤判定しがちな分類：消耗品費（無条件）**

100円ショップは雑貨・文具・食品が幅広く揃うため、店名だけで「消耗品費」と決めると食品中心の購入も誤分類する。

判定の流れ：
1. 品目に文具・小物・雑貨が含まれる → **消耗品費**
2. 品目が食品（菓子・飲料）のみ → **会議費**（茶菓子扱い）または **雑費**（自分用と明らか）
3. 品目不明（合計額のみ） → **消耗品費**（100円ショップのデフォルト用途として）

**デフォルト判定：消耗品費**（品目不明時）

### ケース3：スターバックス 580 円

**誤判定しがちな分類：雑費（自分の食事と見なされる）**

カフェ単独レシートは「自分の休憩」と思いがちだが、本ルールでは：

- カフェ・喫茶店は **会議費** をデフォルトとする（業務利用の打合せ前提）
- 「自分1人の食事として明らかな場合」のみ雑費に落とす（例：ファストフード+ハンバーガーセット1人分が明確）

**判定：会議費**

判断基準：レシート1枚で1人分か複数人分か不明な場合は、店の業態（カフェ・喫茶店）を優先して **会議費** とする。

---

## 出力フォーマット

このスキルが呼び出された結果は、以下の文字列のいずれかとする：

```
会議費 / 消耗品費 / 旅費交通費 / 新聞図書費 / 雑費
```

**5分類以外の文字列を返してはならない**（"食費"、"接待費"、"通信費" 等は禁止）。
